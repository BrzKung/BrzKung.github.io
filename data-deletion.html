  • ลบข้อมูลผู้ใช้

คำแนะนำการลบข้อมูลผู้ใช้ (User Data Deletion)

อัปเดตล่าสุด:

หากคุณต้องการลบข้อมูลส่วนบุคคลที่เก็บไว้จากเกมหรือแอปของ กรุณาทำตามขั้นตอนด้านล่าง

วิธีขอลบข้อมูล

  1. ส่งอีเมลถึงเราที่
  2. ระบุหัวข้ออีเมล: คำขอลบข้อมูลผู้ใช้ / Data Deletion Request
  3. แนบ Facebook User ID หรือชื่อโปรไฟล์ Facebook ที่ใช้ในเกม/แอป
  4. ระบุชื่อเกมหรือแอปที่ต้องการลบข้อมูล (ถ้ามีหลายผลิตภัณฑ์)

ระยะเวลาดำเนินการ

เราจะดำเนินการลบข้อมูลภายใน 30 วันนับจากวันที่ได้รับคำขอที่สมบูรณ์ และแจ้งผลทางอีเมลเมื่อดำเนินการเสร็จสิ้น

ข้อมูลที่จะถูกลบ

  • ข้อมูลโปรไฟล์และตัวระบุผู้ใช้ (เช่น Facebook User ID)
  • คะแนน สถิติ และข้อมูลความคืบหน้าในเกม
  • ข้อมูลการตั้งค่าที่เกี่ยวข้องกับบัญชีของคุณ

ข้อมูลที่อาจเก็บต่อไป

เราอาจเก็บข้อมูลบางส่วนตามที่กฎหมายกำหนด เช่น บันทึกการทำธุรกรรมหรือข้อมูลที่จำเป็นต่อการปฏิบัติตามกฎหมาย
